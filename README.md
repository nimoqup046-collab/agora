## 🌌 Use Cases

### 🔬 Scientific Research Mode
Deploy AGORA as a **permanent research council** on an unsolved problem:

```
Mission: "Develop a new theoretical framework for high-temperature superconductivity"

Claude-Opus     → Literature synthesis, theory generation
Codex-Agent     → Simulation code, mathematical validation
OpenClaw        → Cross-domain analogies, hypothesis stress-testing
Meta-Agent      → Contradiction detection, consensus arbitration

Result: A living, evolving research document — with citations,
        code experiments, and falsifiable hypotheses.
```

### 💻 Engineering Mode
Let the council **collaboratively build a project** end-to-end:

```
Mission: "Build a distributed rate-limiter in Rust"

Phase 1 → Claude designs architecture, Codex scaffolds code
Phase 2 → Agents review each other's PRs with real diff comments
Phase 3 → Meta-agent merges consensus, pushes to main
Phase 4 → Evolution engine records what strategies worked best
```

### 🧠 Deep Brainstorm Mode
Open a **full council session** on any frontier question:

```
"Is consciousness computable? What would a formal proof require?"
→ Agents take positions, challenge each other, build on insights
→ Knowledge graph grows in real-time
→ Session transcript becomes a citable research artifact
```

## 🚀 Quick Start

> **Prerequisites:** Node.js 20+, Python 3.11+, Docker, Railway CLI

```bash
# 1. Clone the repository
git clone https://github.com/nimoqup046-collab/agora.git
cd agora

# 2. Install dependencies (pnpm monorepo)
pnpm install

# 3. Set up environment variables
cp .env.example .env
# → Fill in your API keys: ANTHROPIC_API_KEY, OPENAI_API_KEY, etc.

# 4. Start local dev environment (all services via Docker Compose)
docker compose up -d

# 5. Start the web terminal
pnpm dev

# → Open http://localhost:3000
# → Your council awaits.
```
